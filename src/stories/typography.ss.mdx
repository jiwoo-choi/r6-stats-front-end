# Typography

프로젝트에 사용된 타입 정보입니다.

## Font

`Nunito Sans` 를 사용합니다.

## Weights 

`400(regular)`, `700(bold)`, `800(extrabold)`, `900(black)` 로 나뉘어있습니다.

## Size

`s1~s3 (12~16)`, `m1~m3(20~24)`, `l1~l3 (32~48)`
